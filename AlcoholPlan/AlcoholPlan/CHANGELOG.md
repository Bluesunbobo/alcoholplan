# Changelog - Druk (微醺志)

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-22
### Added
- **Cinematic Core**: High-end "Midnight Tavern" UI with glassmorphic elements and premium animations.
- **BAC Engine**: Real-time Blood Alcohol Concentration calculation based on the Widmark formula.
- **Granular Recording**: Bidirectional calculation for alcohol volume, concentration, and BAC impact.
- **Poster System**: Zero-latency asynchronous rendering of three high-fidelity posters (Moment, Session, Annual Heatmap).
- **Privacy & Compliance**: 7-section bilingual privacy policy and philosophy pages.
- **Permission Guidance**: Intelligent startup photo-library permission check and guidance system.

### Fixed
- Resolved "Black Screen" rendering delays via lazy loading architecture.
- Fixed Swift 6 concurrency warnings and actor-isolation issues.
- Fixed permission alert visibility issues in nested view controllers.
