# Druk 1.0.0 - 《微醺志》功能介绍

《微醺志》(Druk) 是一款结合了电影艺术视角与精密生物算法的饮酒记录与生理模拟应用。它不仅是一个计数的工具，更是一个探索个体与酒精之间平衡关系的“午夜登记册”。

## 1. 核心功能亮点

### 🧪 精密生理模拟引擎 (The Math)
*   **改进型 Widmark 算法**：基于性别、体重、酒精浓度、饮酒量及时间跨度，实时推算血液酒精浓度 (BAC)。
*   **多维代谢模型**：系统会根据用户选择的“代谢速率”阶梯，动态调整酒精排除曲线。
*   **预测模式 (Predictive Mode)**：用户可以在确认饮酒前输入数值，系统将预判摄入后的 BAC 峰值及清醒时间。

### 🎭 电影化人格系统 (The Personas)
*   **身份映射**：内置多种人格角色（如“怀疑论者”、“哲学家”等），每种角色对应特定的代谢背景，让冰冷的数据具备叙事性。
*   **动态交互**：根据不同的醉酒阶段（从“温和”到“狂放”），UI 会自动匹配独特的文学语录与视觉氛围。

### ⚖️ 全球法律管辖区 (Jurisdiction)
*   **动态阈值**：内置全球主流国家的饮酒驾驶 (DUI/DWI) 法定限制标准。
*   **实时对比**：系统会自动将当下的 BAC 估值与所在地法律进行实时对比，提供视觉化的安全预警。

### 📊 年度回顾与社交生成 (Archive & Share)
*   **年度热力图**：通过高质感的网格视图，展示全年的饮酒强度分布，让生活痕迹一目了然。
*   **电影风分享报**：自动生成带有高级排版、代谢曲线、人像剪影及中英双语语录的长图海报。

## 2. 交互体验

*   **午夜序章 (Cinematic Intro)**：每次重新出发，都会以致敬电影《Druk》的沉浸式文字渐入作为开启。
*   **玻璃拟态 (Glassmorphism)**：全 UI 采用层次丰富的磨砂玻璃材质，模拟酒吧灯光在酒杯上的折射感。
*   **触觉反馈**：核心操作均匹配了不同权重的震动脉冲，增强物理触感。
